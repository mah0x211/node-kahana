#node-kahana

an implemented of my own user interface design script.

## Dependency

* ClearSilver 0.10.5

## Installation

    install ClearSilver
	export CPPFLAGS="-I/installed/ClearSilver/header/dir $CPPFLAGS"
    npm install Kahana

## Features

* Observer pattern
